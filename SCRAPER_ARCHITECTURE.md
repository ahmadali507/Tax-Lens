### 2. Scheduling Process

```
npm run scraper:start
      │
      ├─► Start Background Scheduler
      │   (3,600,000 ms interval = 1 hour)
      │
      ├─► Run Scraper Immediately
      │   │
      │   └─► Generate projects.ts ✓
      │
      ├─► Schedule Next Run
      │   (1 hour later)
      │   │
      │   └─► Run Scraper
      │       └─► Update projects.ts ✓
      │
      ├─► Continue Every Hour
      │   (infinite loop)
      │   │
      │   └─► Update projects.ts ✓
      │
      └─► Stop on Ctrl+C
          (Clear interval)
```

```
User Opens Browser
      │
      ├─► GET /projects
      │   │

      │       │   • Show name

      │       │   • Show budget details
      │       │   • Show status badge
      │       │
      │       └─► Display in 3-column grid
      │
      └─► Display 9 Real Projects ✓
```

## Commands Reference

```bash
# Development
npm run dev                    # Start dev server on localhost:3000

# Scraper Commands
npm run scraper:once         # Run scraper once
npm run scraper:start        # Start auto-scheduler (every 1 hour)
npm run scraper:status       # Check scheduler status
npm run scraper:help         # Show CLI help

# Build & Production
npm run build                # Build for production
npm run start                # Start production server

# Linting
npm lint                     # Run ESLint
```
